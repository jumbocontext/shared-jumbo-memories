# Leave the code cleaner than you found it (Boy Scout Rule)
Reasoning: Small continuous improvements compound over time and prevent gradual rot from building into large-scale rewrites.

```bash
jumbo invariant add --title "Leave the code cleaner than you found it (Boy Scout Rule)" --description "Whenever you touch a file, leave it in better shape than you found it. Fix the obvious inconsistency, rename the unclear variable, delete the dead comment or unused import. No change is too small. Version control ensures nothing is lost — there is no excuse for leaving known mess behind."
```
