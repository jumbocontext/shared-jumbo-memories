# CLI help must be discoverable
Reasoning: Users should never need to read source code to understand how to invoke a command. Good --help output lowers friction for first-time users and people returning to infrequent commands.

```bash
jumbo guideline add --category cli --title "CLI help must be discoverable" --description "Every command and flag must be documented via --help. Users should never need to read source code to understand how to invoke a command."
```
