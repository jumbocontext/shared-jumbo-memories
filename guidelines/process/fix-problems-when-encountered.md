# Fix problems when encountered
Reasoning: Small fixes made while context is fresh are cheaper than deferred cleanup work that may never be prioritized.

```bash
jumbo guideline add --category process --title "Fix problems when encountered" --description "When you notice naming inconsistencies, outdated patterns, or other issues while working on a task, fix them as part of the current work rather than deferring to a separate task." --rationale "Small fixes made while context is fresh are cheaper than deferred cleanup work that may never be prioritized."
```
