# Provide web documentation
Reasoning: In-CLI help covers invocation but cannot replace a README for communicating purpose, installation, and context to someone evaluating or onboarding to a tool or library.

```bash
jumbo guideline add --category documentation --title "Provide web documentation" --description "Every tool or library must have a README that covers its purpose, installation, and basic usage. Users should not need to clone or run code to understand what a project does or how to get started."
```
